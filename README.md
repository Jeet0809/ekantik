# Ekantik - Niyam Tracker

Goal: Ship a Python CLI with SQLite by Dec 20, 2025.

**Tech Stack**

- Python 3.12
- SQLite
- Ubuntu 24.04 (WSL2)
- Git + GitHub (SSH)

**Features (Dec 2025 Target)**

- Add niyam entry
- View niyam history
- Calculate streaks
- Persistent storage in SQLite
- Optional: backup to AWS S3 (later)

This is the foundation module for the Ekantik OS.
